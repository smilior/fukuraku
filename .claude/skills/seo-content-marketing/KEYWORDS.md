```
「freee 副業 高い」

「確定申告 アプリ 比較 2025」

```

**比較記事のテンプレート：**
```
タイトル：「副業サラリーマンの確定申告アプリ比較【2025年版】
           freee vs マネーフォワード vs 副楽」

4. 副楽の特徴と向いている人
5. 結論：副業サラリーマンには副楽がおすすめ
```

## キーワード選定の考え方

**検索意図の4分類（SEOの基本）：**

| 検索意図 | 例 | 副楽での対策 |
|---------|-----|-----------------|
| 情報型（Know） | 「副業 確定申告 やり方」 | ブログ記事で集客 |
| 調査型（Investigate） | 「確定申告 アプリ 比較」 | 比較記事 + LP |
| 取引型（Do/Buy） | 「副楽 登録」 | LP最適化 |
| 案内型（Go） | 「副楽 ログイン」 | ログインページ最適化 |

**戦略：情報型 → 調査型 → 取引型 の導線を作る**

```
ブログ記事「副業 確定申告 やり方」（情報型）
  ↓ 記事内CTA
比較記事「確定申告 アプリ 比較」（調査型）
  ↓ 副楽が最適という結論
LP → 登録（取引型）
```

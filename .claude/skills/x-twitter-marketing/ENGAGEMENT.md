# エンゲージメント増加施策

## 5.1 リプライ返信のルール

**タイミング：**
- 投稿後1時間以内のリプライには最優先で返信
- アルゴリズム上、早期のエンゲージメントが重要
- 自分のリプライもエンゲージメントとしてカウントされる

**返信のコツ：**
- 質問には必ず回答する
- 「ありがとう」だけでなく、追加情報を添える
- 相手の投稿にも積極的にいいね・リプライする
- 批判的なリプライには冷静に対応（感情的にならない）

## 5.2 副業系・税金系インフルエンサーへのアプローチ

**アプローチ対象のカテゴリ：**
- 副業系インフルエンサー（フォロワー1万人以上）
- 税理士アカウント（税金の解説をしている人）
- 個人開発者（同じ境遇の仲間）
- ファイナンシャルプランナー

**アプローチ方法（段階的）：**
1. まずフォローして投稿を日常的にいいね・RTする（2週間）
2. 有益なリプライを送って存在を認知してもらう（2週間）
3. DMで自己紹介とアプリの紹介（謙虚に）
4. ベータテストへの参加依頼
5. コラボ企画の提案（共同スペース、相互紹介等）

**DMテンプレート：**
```
○○さん、はじめまして。
副業の確定申告支援アプリ「副楽」を個人開発している[名前]です。

○○さんの副業に関する発信をいつも参考にしています。
特に[具体的な投稿]の内容は大変勉強になりました。

現在、副業サラリーマン向けに
レシートを撮るだけでAIが経費を自動分類するアプリを開発しており、
もしよろしければベータテスターとしてフィードバックをいただけないでしょうか。

○○さんのフォロワーさんにも役立つサービスだと考えています。
ご検討いただけると幸いです。
```

## 5.3 コミュニティ参加

**参加すべきコミュニティ：**

1. **Xのスペース**
   - 副業系スペース（毎週開催されているものを探す）
   - 個人開発者のスペース
   - 確定申告シーズンの特別スペース

2. **Discord/Slack**
   - 副業系コミュニティ
   - 個人開発者コミュニティ（Indie Hackers Japan等）
   - Next.js/Supabase コミュニティ

3. **note/Zenn のコミュニティ**
   - 副業カテゴリの記事にコメント
   - 技術記事でのクロスプロモーション

## 5.4 相互フォロー・相互RT

**やるべきこと：**
- 個人開発者同士の相互応援（#個人開発 タグ）
- 副業系アカウントとの相互いいね
- 確定申告シーズンの税理士アカウントの投稿をRTしてコメント

**やってはいけないこと：**
- 無差別な相互フォロー依頼
- スパム的なリプライ
- 過度な自サービスの宣伝（リプ欄での宣伝は控える）

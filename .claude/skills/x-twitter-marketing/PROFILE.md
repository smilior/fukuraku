# アカウント設定・プロフィール最適化

## 1.1 アカウントの基本設計

**ポジショニング：**
副楽の公式アカウントとして運用するか、開発者個人のアカウントとして運用するかを決めます。

**推奨：開発者個人アカウントでの運用**
- 個人開発の親近感・ストーリー性がバズりやすい
- 「副業×確定申告×個人開発」という独自ポジション
- 副楽公式アカウントは別途作成し、個人アカウントからリポストする運用

**アカウント名の候補：**
- 個人アカウント：「@[名前] | 副業×確定申告アプリ開発中」
- 公式アカウント：「@fukuraku_app」

## 1.3 ヘッダー画像の推奨内容

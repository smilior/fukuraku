# 曜日別投稿テーマ・テンプレート集

## 投稿の基本ルール

- 1日1〜2投稿（朝7-8時 or 昼12-13時 or 夜20-22時）
- 画像またはスレッド形式で投稿するとインプレッション2-3倍
- ハッシュタグは1-2個まで（多すぎると逆効果）
- CTAは3投稿に1回程度（頻繁すぎると敬遠される）

## 月曜日：副業×税金Tips

副業サラリーマンが知っておくべき税金の知識をわかりやすく解説。

**投稿例1：**
```

収入 - 経費 = 所得

つまり、売上が30万円でも
経費が15万円なら
所得は15万円で申告不要。

経費をきちんと記録するだけで
申告が不要になるケースもある。

#副業 #確定申告
```

**投稿例2：**
```

- パソコン、モニター
- インターネット回線（按分）

- セミナー参加費

- 文房具、消耗品

でも「按分」ってわかりにくいよね。
```

**投稿例3：**
```

よくある間違い：
1. 経費になるものを見落としてる

4. 年末調整済みだから確定申告不要だと思ってる
```

**投稿例4：**
```

「メルカリの売上って確定申告必要？」

- 不用品の販売 → 基本的に非課税
```

## 木曜日：Q&A形式

よくある質問に答える形式で投稿。

**投稿例1：**
```
Q. 副業の収入が20万円以下なら確定申告は不要？

A. 所得税の確定申告は不要です。
ただし、住民税の申告は必要です。

これ、意外と知らない人多い。
住民税の申告をしないと、
自治体から連絡が来ることも。

市区町村の窓口で簡単に申告できます。
```

**投稿例2：**
```
Q. 副業禁止の会社だけど、確定申告したらバレる？

A. 確定申告の方法次第でバレにくくできます。

ポイントは住民税を「自分で納付」にすること。

ただし：
- ふるさと納税のワンストップ特例は使わない
- 医療費控除と一緒に申告する場合は注意
- 100%バレないとは言い切れない

詳しくは税理士に相談を推奨。
```

**投稿例3：**
```
Q. クラウドソーシングの源泉徴収って何？

A. 報酬から事前に税金が引かれる仕組み。

例：報酬10万円の場合
源泉徴収額：10,210円
手取り：89,790円

確定申告すると、
この源泉徴収分が還付される可能性あり。

特に経費が多い人は確定申告した方がお得。
```

**投稿例4：**
```
Q. 家賃って経費にできる？

A. 自宅で副業してるなら「按分」で一部経費にできます。

例：
家賃10万円
作業スペース：部屋全体の20%
→ 月2万円が経費

按分の根拠は
- 面積比
- 使用時間比
のどちらかで計算。

領収書（振込記録）を保存しておくこと。
```

**投稿例5：**
```
Q. 副業で赤字が出た場合はどうなる？

A. 所得の種類によります。

- 雑所得の赤字 → 他の所得と損益通算できない
- 事業所得の赤字 → 給与所得と損益通算可能

つまり、事業所得にできれば
副業の赤字で給与の税金が減る可能性あり。

ただし「事業所得」と認められるには
継続性・反復性が必要。
```

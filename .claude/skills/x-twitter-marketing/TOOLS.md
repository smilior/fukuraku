# ツールと自動化

## 7.1 投稿の予約・管理ツール

**Buffer（推奨）：**
- 無料プラン：3チャンネル、各チャンネル10投稿まで予約可能
- 月額$6〜：より多くの予約と分析機能
- 使い方：
  1. Bufferにアカウントを作成
  2. Xアカウントを連携
  3. 投稿スケジュールを設定（朝8時、夜21時等）
  4. 週末にまとめて1週間分の投稿を予約

**Hootsuite（代替）：**
- 無料プラン：2チャンネル
- 複数SNSを同時管理する場合に有用

**SocialDog（X特化）：**
- X（Twitter）に特化した分析・予約ツール
- フォロワー分析が充実
- 日本語UIで使いやすい

## 7.2 Notion での投稿ネタ管理

**データベース設計：**

| プロパティ | タイプ | 内容 |
|-----------|--------|------|
| タイトル | タイトル | 投稿の概要 |
| カテゴリ | セレクト | Tips/開発進捗/あるある/Q&A/裏話/まとめ |
| ステータス | セレクト | ネタ/下書き/予約済み/投稿済み |
| 投稿予定日 | 日付 | |
| 投稿本文 | テキスト | 実際の投稿文 |
| 画像 | ファイル | 添付画像 |
| インプレッション | 数値 | 投稿後に記録 |
| エンゲージメント率 | 数値 | 投稿後に記録 |
| メモ | テキスト | 気づいたこと |

**運用フロー：**
1. ネタを思いついたらNotionに追加（ステータス：ネタ）
2. 週末に1週間分の投稿を下書き（ステータス：下書き）
3. Bufferに登録して予約（ステータス：予約済み）
4. 投稿されたらパフォーマンスを記録（ステータス：投稿済み）

## 7.4 投稿のリサイクル

反応が良かった投稿は、一定期間後（2-3ヶ月）にリライトして再投稿できます。

**リサイクルのルール：**
- 完全なコピーはNG（フォロワーに気づかれる）
- 角度を変えてリライトする
- 新しい情報やデータを追加する
- 画像を変更する

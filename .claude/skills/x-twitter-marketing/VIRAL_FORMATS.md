# バイラルを狙う投稿フォーマット

## 3.1 数字入りリスト型

数字が入った投稿はクリック率が高く、保存されやすい。

**フォーマット：**
```
[ターゲット]が知らない[テーマ]の[数字]選
```

**具体例：**
```
副業サラリーマンが知らない税金の真実5選：

1. 20万円以下でも住民税は申告必要
2. 家賃の一部は経費にできる
3. 交通系ICの履歴も経費の証拠になる
4. 源泉徴収されてると還付金がもらえるかも
5. 確定申告しないと「無申告加算税」がかかる

知らないと損する。保存推奨。
```

```
個人開発で月1万円稼ぐまでにやったこと7選：

1. 自分の課題をアプリにした
2. 最小限の機能でリリースした
3. X（Twitter）で毎日発信した
4. ベータユーザーの声を聞いた
5. 料金は最初から有料にした
6. 無料プランとの差別化を明確にした
7. 確定申告シーズンに集中マーケした
```

## 3.2 Before/After型

変化の幅が大きいほどインパクトがある。

**フォーマット：**
```
Before: [ネガティブな状態]
After: [ポジティブな状態]
```

**具体例：**
```
確定申告の作業時間：

Before：30時間（レシート整理、Excel入力、計算...）
After：3分（レシート撮影、AI自動分類、完了）

副楽を作った理由はこれ。
```

```
副業の経費管理：

Before：
- レシートが財布に溜まる
- 月末にまとめてExcelに入力
- 分類がわからなくて毎回検索
- 合計額が合わなくて深夜に格闘

After：
- レシートを撮る（3秒）
- AIが自動分類（即時）
- ダッシュボードで一覧確認
- 20万円ラインを自動監視
```

## 3.3 問いかけ型

フォロワーに考えさせる投稿はリプライが増えやすい。

**フォーマット：**
```
[ターゲットの共感を呼ぶ質問]？
```

**具体例：**
```
副業で月5万稼いだら確定申告は必要？

リプで答えてみてください。

正解は...
```

```
副業の確定申告、何時間かかった？

去年の私は30時間でした。

皆さんはどのくらい？
リプで教えてください。
```

```
freeeとマネーフォワード、
副業の確定申告にはどっちがいい？

...と思ってたけど、
副業サラリーマンにはどっちもオーバースペックだった話。
```

## 3.4 ネタバレ型

「結果↓」で続きを読ませる。

**フォーマット：**
```
[行動]した結果↓
```

**具体例：**
```
AIにレシートを読み込ませた結果↓

入力：コンビニのレシート写真

AIの回答：
- 日付：2024年1月15日
- 店名：セブンイレブン○○店
- 金額：1,234円
- 品目：コーヒー、サンドイッチ、電池
- 経費分類：「会議費」（コーヒー）、「消耗品費」（電池）

精度すごくない？
```

```
副業の確定申告を
AIアプリに丸投げしてみた結果↓

所要時間：3分
経費の分類：自動
合計金額：自動計算
申告要否：「申告不要です（所得18万円）」

え、去年30時間かけたのに...
```

## 3.5 共感型

「わかる」と思わせてリツイートを誘発。

**フォーマット：**
```
[共感できる状況]な人RT
```

**具体例：**
```
確定申告の季節、freeeを開いて閉じた人RT
```

```
副業始めたけど確定申告のことを何も考えてなかった人RT
```

```
「副業の経費ってどこまでOK？」を
Google検索したことがある人RT
```

## 3.6 意外性型

常識を覆す情報でインパクトを与える。

```
副業の確定申告、
実は「しない方が損」なケースがある。

源泉徴収されてる人は
確定申告すると税金が戻ってくる可能性が高い。

特にクラウドソーシングで働いてる人、
報酬から10.21%引かれてるの知ってる？

確定申告すれば数万円戻ってくるかも。
```

# 確定申告シーズン（1-3月）集中投稿プラン

## 4.1 1月：意識喚起フェーズ

**テーマ：「そろそろ準備始めよう」**

1月の投稿カレンダー：

**第1週：年始の意識喚起**
- 1/1：「今年こそ確定申告を楽にする方法」
- 1/2：「副業サラリーマンの新年やること3つ」
- 1/3：「去年の確定申告、何時間かかった？」（問いかけ型）

**第2週：準備のススメ**
- 1/6：「確定申告の準備チェックリスト」（まとめ型）
- 1/8：「源泉徴収票が届いたらやること」
- 1/10：「レシート整理を今からやるべき理由」

**第3週：ツール紹介**
- 1/13：「確定申告アプリ比較（freee, MF, 副楽）」
- 1/15：「副業サラリーマンにfreeeは高すぎる問題」
- 1/17：「レシート撮るだけで経費管理できるアプリ作った」

**第4週：危機感の喚起**
- 1/20：「確定申告まであと2ヶ月。準備してる？」
- 1/22：「無申告のリスクを知ってますか？」
- 1/24：「今からでも間に合う確定申告準備」

## 4.2 2月：解決策提示フェーズ

**テーマ：「副楽で解決しよう」**

**第1週：問題の深堀り**
- 2/3：「確定申告で一番面倒なのは何？」（問いかけ型）
- 2/5：「経費の分類、誰かやってくれないかな」（共感型）

**第2週：解決策の提示**
- 2/10：「副楽でレシートを撮ってみた」（ネタバレ型）
- 2/12：「確定申告の作業時間 Before/After」（Before/After型）

**第3週：使い方紹介**
- 2/17：「副楽の使い方3ステップ」
- 2/19：「20万円ラインの自動監視機能が便利すぎる」

**第4週：ユーザーの声**
- 2/24：「ベータユーザーの声を紹介」
- 2/26：「副楽で確定申告の準備が終わった人の報告」

## 4.3 3月：締切カウントダウンフェーズ

**テーマ：「まだ間に合う！」**

- 3/1：「確定申告期限まであと15日」
- 3/5：「まだ確定申告してない人、今すぐ副楽で」
- 3/8：「確定申告期限まであと1週間」（緊急感）
- 3/10：「あと5日。今からでも副楽なら3分で準備完了」
- 3/12：「あと3日。最後のチャンス」
- 3/14：「明日が期限。まだの人は副楽へ」
- 3/15：「本日期限！お疲れ様でした」
- 3/16：「確定申告お疲れ様。来年は副楽で楽にしよう」

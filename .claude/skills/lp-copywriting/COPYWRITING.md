4. **「週末を確定申告に使うのは、もうやめよう。」**

5. **「寝る前の3分で、確定申告の準備が終わる。」**
   - 簡単さと時間の短さを具体的に

   - 最も多い疑問に即答する

## ターゲット（副業サラリーマン）に刺さるコピーパターン

### ペルソナの定義

**メインターゲット：**
- 年齢：25-45歳
- 職業：会社員（副業をしている）
- 副業の種類：プログラミング、ライティング、デザイン、ブログ、物販
- 年間副業収入：20万円〜100万円
- ITリテラシー：中程度（スマホアプリは日常的に使用）
- 課題：確定申告の知識がない、時間がない、面倒くさい

**サブターゲット：**
- 副業を始めたばかりの人（確定申告の存在を知ったばかり）
- freee/マネーフォワードが複雑すぎて挫折した人
- 来年こそちゃんと経費管理しようと思っている人

### 時間節約パターン

ターゲットは忙しい会社員。時間の節約は最も刺さるメッセージ。

```
「30時間の作業が3分に」
「週末を確定申告に使わなくていい」
「レシートを撮る3秒で経費登録完了」
「毎月5分の管理で、3月の地獄を回避」
「今日から始めれば、来年の確定申告は3分で終わる」
「通勤電車の中で経費管理が完了」
「昼休みの5分で1ヶ月分の経費を整理」
```

**使用シーン：**
- Heroセクションのサブコピー
- 機能紹介の見出し
- CTAの周辺テキスト

### 不安解消パターン

確定申告に対する漠然とした不安を持つユーザーに刺さる。

```
「申告漏れのリスクをゼロに」
```

## CTA文言バリエーション

### メインCTA（登録誘導）

**推奨文言：**
```
「無料で始める」          ← 最もシンプルで効果的
「今すぐ無料で試す」      ← 即時性を強調
「30秒で登録完了」        ← 簡単さを強調
「フリープランで始める」   ← 無料であることを明確に
```

**避けるべき文言：**
```
「登録する」              ← 心理的ハードルが高い
「購入する」              ← 有料の印象
「申し込む」              ← 堅い印象
「送信する」              ← 何を送信するのか不明
```

### サブテキスト（安心材料）

CTAボタンの下に表示する安心材料テキスト。

```
「クレジットカード不要」
「30秒で登録完了」
「いつでも解約OK」
「メールアドレスだけで始められます」
「XXX人が利用中」
「フリープランはずっと無料」
```

**組み合わせ例：**
```
[無料で始める]  ← ボタン
クレジットカード不要 / 30秒で登録完了
```

### セクション別CTA文言

**Heroセクション：**
```
「無料で始める」+ 「クレジットカード不要」
```

**機能紹介セクション直下：**
```
「レシートを撮ってみる」+ 「まずはフリープランで体験」
```

**料金プランセクション：**

**最終CTA（ページ末尾）：**

```
+ 「フリープランはずっと無料。まずは試してみてください。」
```

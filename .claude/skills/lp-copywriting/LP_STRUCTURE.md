# LP基本構成テンプレート（Hero〜CTA）

## LP全体の構成フロー

SaaSのLPは以下の順序で構成します。各セクションの役割を明確にし、
ユーザーを自然にCTAへ導きます。

```
1. Hero（ファーストビュー）     → 興味を引く
2. 問題提起                    → 共感を得る
3. 解決策の提示                → 希望を持たせる
4. 機能紹介                    → 信頼を構築する
5. 使い方（3ステップ）          → 簡単さをアピール
6. 社会的証明                  → 安心感を与える
7. 料金プラン                  → 選択肢を提示
8. FAQ                         → 不安を解消
9. 最終CTA                     → 行動を促す
10. フッター
```

---

## Heroセクション

**目的：** 3秒以内にユーザーの注意を引き、何のサービスかを伝える。

**構成要素：**
- キャッチコピー（H1）：1行、16文字以内が理想
- サブコピー（H2）：2行以内で具体的な価値を伝える
- CTAボタン：1つ（「無料で始める」等）
- サブテキスト：「クレジットカード不要」等の安心材料
- ヒーロー画像またはアプリのスクリーンショット

**副楽向けHeroコピー例：**

案A（シンプル）：
```
H1: レシートを撮るだけ。
H2: 副業の確定申告、AIにおまかせ。
CTA: 無料で始める
サブ: クレジットカード不要 / 30秒で登録完了
```

案B（課題解決型）：
```
H1: 副業の確定申告、もう悩まない。
H2: レシートを撮るだけでAIが経費を自動分類。
    20万円ラインの監視もおまかせ。
CTA: 今すぐ無料で試す
サブ: 2,800人が利用中（ベータ含む）
```

案C（数字インパクト型）：
```
H1: 確定申告の準備、30時間→3分。
H2: 副業サラリーマン専用のAI確定申告アプリ。
CTA: 無料プランで始める
サブ: クレジットカード不要 / いつでも解約OK
```

## 問題提起セクション

**目的：** ターゲットが抱える問題・不安を言語化し、「これは自分のことだ」と思わせる。

```
セクションタイトル：
「副業の確定申告、こんなことで悩んでいませんか？」

課題リスト：
- レシートが溜まっていくばかりで整理する時間がない
- 何が経費にできるのかよくわからない
- 副業の所得が20万円を超えたか計算するのが面倒
- freeeやマネーフォワードは機能が多すぎて使いこなせない
- 確定申告の時期になると毎年パニックになる
- 会社にバレないか不安で調べるだけで疲れる
```

ストーリー型の問題提起：
```
毎年3月、あなたはこうなっていませんか？

財布の中のレシートを机に広げる。
Excelを開く。
国税庁のサイトを検索する。
「副業 経費 何が」で検索する。
途中で挫折する。

翌日、同じことを繰り返す。
そして気づけば3月14日の深夜...

もう、そんな確定申告は終わりにしませんか？
```

## 解決策セクション

**目的：** 副楽がどのように問題を解決するかを簡潔に伝える。

```
セクションタイトル：「副楽なら、レシートを撮るだけ。」
サブコピー：「AIが経費を自動分類し、確定申告の準備を3分で完了します。」

3つの価値提案：
1. レシートを撮るだけで経費を自動記録
   → もうExcelに手入力する必要はありません。

2. AIが経費を自動で分類
   → 「これは交通費？通信費？」と迷う時間をゼロに。

3. 20万円ラインを自動で監視
   → 確定申告が必要かどうか、いつでも確認できます。
```

## 機能紹介セクション

**目的：** 具体的な機能を3-4つに絞ってビジュアルで紹介する。

**構成パターン：** 画像 + コピーの交互配置（ジグザグレイアウト）

**機能1：AI経費自動分類**
```
見出し：レシートを撮るだけ。AIが全部やります。
説明：スマートフォンでレシートを撮影するだけで、
      AIが日付・金額・品目を読み取り、
      適切な経費カテゴリに自動分類します。
      精度94%。あなたの手間はたった3秒です。
[スクリーンショット：レシート撮影 → 自動分類結果]
```

**機能2：20万円ライン自動監視**
```
見出し：「確定申告、必要？」がひと目でわかる。
説明：副業の所得（収入-経費）をリアルタイムで計算。
      20万円ラインに近づくとアラートでお知らせ。
      「いつの間にか超えてた...」を防ぎます。
[スクリーンショット：ダッシュボードのグラフ + アラート]
```

**機能3：確定申告書類の自動生成**
```
見出し：確定申告に必要な情報、自動で集計。
説明：年間の収入と経費を自動集計し、
      確定申告に必要な数字を一画面にまとめます。
      e-Taxへの転記もコピーするだけ。
[スクリーンショット：年間集計画面]
```

**機能4：データの安全管理**
```
見出し：あなたの大切なデータ、しっかり守ります。
説明：通信はTLS 1.3で暗号化。データはSupabaseの
      東京リージョンに保存。
      ユーザーごとにデータを厳密に分離しています。
[セキュリティアイコンの図]
```

## 使い方セクション（3ステップ）

**目的：** 使い方がいかに簡単かを3ステップで伝える。

```
セクションタイトル：「3ステップで確定申告の準備完了」

Step 1: レシートを撮る
  スマートフォンでレシートを撮影するだけ。
  まとめて撮っても1枚ずつ撮ってもOK。

Step 2: AIが自動分類
  AIが日付、金額、品目を読み取り、
  経費カテゴリに自動分類します。
  あなたは確認するだけ。

Step 3: 結果を確認
  ダッシュボードで所得と経費の状況を確認。
  確定申告が必要かどうか、ひと目でわかります。
```

**目的：** 他のユーザーが使っていること、満足していることを示す。

**ベータユーザーの声テンプレート：**
```
「毎年Excelで管理していたレシートが、撮るだけでOKになった。
 感動しました。」
 - 30代 会社員 / フリーランスライター（副業歴3年）

「20万円ラインの自動監視が便利。いちいち計算しなくていい。」
 - 20代 会社員 / プログラミング講師（副業歴1年）

「freeeが難しくて挫折した自分でも使えた。」
 - 40代 会社員 / ブロガー（副業歴5年）
```

**数字で示す社会的証明：**
```
- XXX人が利用中
- 平均作業時間 3分
- ユーザー満足度 XX%
- レシート読取精度 94%
```

数字は正確な値のみを使用。誇張は景品表示法違反のリスクあり。

## FAQセクション

**目的：** ユーザーの不安・疑問を先回りして解消する。

```
Q. 本当に無料で使えますか？
A. はい。フリープランは無料で、月5枚までのレシート登録と
   基本的な経費管理機能をご利用いただけます。
   クレジットカードの登録は不要です。

Q. 税理士の代わりになりますか？
A. いいえ。副楽は経費の記録・管理を効率化するツールです。
   個別の税務相談は税理士にご依頼ください。
   AIの分析結果は参考情報としてご活用ください。

Q. レシートの画像データは安全ですか？
A. はい。全データはTLS 1.3で暗号化して送信し、
   Supabaseの東京リージョンに保存されます。
   ユーザーごとにデータは厳密に分離されています。

Q. 確定申告書の作成まで自動でやってくれますか？
A. 確定申告に必要な金額（収入・経費の合計、所得額）を
   自動で集計し、一覧表示します。
   確定申告書自体の作成はe-Tax等で行っていただきますが、
   必要な数字のコピー&ペーストで転記できます。

Q. 解約は簡単にできますか？
A. はい。設定画面から「プランを解約する」ボタンで
   いつでも解約できます。解約手続きは1分以内に完了します。
   解約後も現在の課金期間の終了日まで利用可能です。

Q. 他の確定申告ソフトとの違いは？
A. freeeやマネーフォワードは本格的な会計ソフトですが、
   副楽は「副業サラリーマン」に特化しています。
   必要な機能だけをシンプルに提供し、
   料金も月額980円と手頃です。

Q. 会社に副業がバレませんか？
A. 副楽の利用自体が会社にバレることはありません。
   確定申告時に住民税を「自分で納付」にすることで
   副業がバレるリスクを軽減できます。
   設定方法はアプリ内のガイドで解説しています。

Q. データのエクスポートはできますか？
A. はい。経費データをCSV形式でエクスポートできます。
   他の会計ソフトへの移行にも対応しています。
```

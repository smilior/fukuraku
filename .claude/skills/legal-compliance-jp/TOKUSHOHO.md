# 特定商取引法に基づく表示

## 特定商取引法（特商法）の適用範囲

インターネットを通じて有料サービスを提供する場合、特定商取引法の
「通信販売」に該当し、以下の表示が必要です。

**適用条件（副楽の場合）：**
- 有料プラン（月額サブスクリプション）を提供する場合に適用
- 無料プランのみの場合でも、将来の有料化を見据えて準備しておくことを推奨

---

## 必須記載事項一覧

特定商取引法第11条に基づく表示事項：

| 項目 | 記載内容 | 備考 |
|------|---------|------|
| 販売事業者名 | [個人名またはサービス名] | 個人事業主の場合は本名 |
| 代表者名 | [氏名] | 法人の場合は代表取締役 |
| 所在地 | [住所] | 後述の非公開対応参照 |
| 電話番号 | [電話番号] | 後述の非公開対応参照 |
| メールアドレス | [メールアドレス] | 必須公開 |
| 販売価格 | 月額XXX円（税込） | 税込価格で表示（総額表示義務） |
| 支払方法 | クレジットカード（Stripe） | 対応ブランドも記載推奨 |
| 支払時期 | 登録時即時決済、以後毎月自動課金 | サブスクの場合は自動更新を明記 |
| サービス提供時期 | 決済完了後直ちに利用可能 | |
| 返品・キャンセル | デジタルコンテンツのため返品不可。解約は次回課金日前まで。 | |
| 解約方法 | アカウント設定画面から解約可能 | 具体的な手順を明記 |
| 動作環境 | Chrome/Safari/Edge最新版、iOS/Android | |

## 個人運営の場合の住所・電話番号の非公開対応

個人事業主の場合、自宅住所や個人電話番号の公開に抵抗があるケースが多いです。

**対応方法：**

1. **バーチャルオフィスの利用**
   - 月額数千円でビジネス住所を取得可能
   - 法人登記にも使える住所を選ぶと将来的に安心
   - 推奨サービス例：GMOオフィスサポート、レゾナンス、Karigo等

2. **電話番号の対応**
   - 050番号（IP電話）を取得して掲載
   - 「請求があった場合に遅滞なく開示」と記載し、メールアドレスのみ公開する方法も
     特商法上は認められている（ただし推奨は電話番号の掲載）

3. **「請求により遅滞なく開示」の活用**
   - 住所・電話番号について「請求があれば遅滞なく開示します」と記載
   - ただし、この方法を取る場合もメールアドレスは必ず公開必須
   - 消費者からの問い合わせに確実に対応できる体制が必要

## サブスクリプション解約方法の明示

2022年6月施行の改正特商法により、サブスクリプション（定期購入）の解約方法を
わかりやすく表示することが義務付けられました。

**解約に関する必須表示：**
- 解約の申し出先（設定画面のURL、問い合わせメール）
- 解約の方法（具体的な手順をステップで記載）
- 解約の条件（次回課金日前日までに手続きが必要等）
- 解約後のデータの取扱い

**解約手順の記載例：**
```
【解約方法】
1. 副楽にログインします
2. 右上のアイコンから「設定」を選択します
3. 「プラン管理」タブを選択します
4. 「プランを解約する」ボタンをクリックします
5. 確認画面で「解約する」を選択します
6. 解約完了メールが届きます

※ 解約後も現在の課金期間の終了日まではサービスをご利用いただけます。
※ 解約後30日以内であれば、設定画面からプランを再開できます。
```

## 副楽向け特商法表示ひな形

```
特定商取引法に基づく表示

販売事業者：[氏名または屋号]
代表者：[氏名]
所在地：ご請求いただければ遅滞なく開示いたします
電話番号：ご請求いただければ遅滞なく開示いたします
メールアドレス：[メールアドレス]
販売URL：https://fukuraku.app（仮）

【販売価格】
- フリープラン：無料
- プロプラン：月額980円（税込）
- シーズンパス：2,980円/年（税込、確定申告シーズン限定）

※ 上記は予定価格であり、変更される場合があります。

【支払方法】
クレジットカード（VISA、Mastercard、American Express、JCB）
※ Stripe社の決済システムを利用しています

【支払時期】
登録時に即時決済。以後、毎月同日に自動課金。

【サービス提供時期】
決済完了後、直ちにご利用いただけます。

【返品・キャンセル】
デジタルサービスの性質上、購入後の返品・返金はお受けしておりません。
ただし、サービスに重大な不具合があった場合は個別にご対応いたします。

【解約方法】
アカウント設定画面の「プラン管理」より、いつでも解約いただけます。
解約後も、現在の課金期間の終了日まではサービスをご利用いただけます。

【動作環境】
- ブラウザ：Google Chrome、Safari、Microsoft Edge（各最新版）
- スマートフォン：iOS 15以上、Android 12以上
- インターネット接続が必要です
```

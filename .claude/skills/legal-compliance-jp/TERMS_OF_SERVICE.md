## 税務関連サービスの重要注意点

- 税理士法第2条に基づき、個別具体的な税務相談・税務代理は税理士の独占業務
- アプリが提供するのは「情報整理ツール」であり「税務相談」ではないことを明記
- AIによる分析結果は参考情報であり、最終判断はユーザー自身（または税理士）が行うことを明記

## 利用規約の実装ポイント

**同意取得のUI実装：**
- 会員登録時に利用規約とプライバシーポリシーへのリンクを表示
- チェックボックスで明示的な同意を取得
- 同意日時をデータベースに記録（`agreed_at` カラム）
- 規約変更時は再度同意を求めるフローを用意

**Supabase での同意記録テーブル例：**
```sql
CREATE TABLE user_agreements (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id) NOT NULL,
  agreement_type TEXT NOT NULL, -- 'terms' or 'privacy'
  version TEXT NOT NULL,        -- '2024-01-01' など
  agreed_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
  ip_address INET,
  user_agent TEXT
);
```

## 未成年者利用への対応

### 年齢確認の要否

**副業確定申告アプリの場合：**
- 主なターゲットは成人（副業サラリーマン）
- ただし、未成年者でもアルバイト等の収入がある場合は利用の可能性あり
- 民法上、未成年者の法律行為は原則として法定代理人の同意が必要（第5条）

**推奨対応：**
- 利用規約で「18歳未満の方は、保護者の同意を得た上でご利用ください」と明記
- 有料プランの決済時には18歳以上であることの確認を推奨
- 年齢確認の方法は自己申告（チェックボックス）で可

### 保護者の同意

**消費者契約法・民法との関係：**
- 未成年者が保護者の同意なく行った契約は取り消し可能
- ただし、「小遣いの範囲内」の少額決済は取消不可とされるケースもある
- 無料プランの利用は契約の取消リスクは低い

**実装方針：**
- 無料プラン：特別な年齢確認なしで利用可能
- 有料プラン：利用規約の同意時に「18歳以上である、または保護者の同意を得ている」
  ことのチェックボックスを設置
- 保護者からの問い合わせ・取消要求には誠実に対応

## 注意事項

本テンプレートで提供する情報は一般的な参考情報であり、法的助言ではありません。
実際の利用規約作成にあたっては、弁護士等の法律専門家に確認することを強く推奨します。

特に以下の点については専門家への相談を推奨します：
- 免責条項の有効性の確認
- 消費者契約法上の不当条項に該当しないかの確認
- 税理士法との抵触リスクの評価

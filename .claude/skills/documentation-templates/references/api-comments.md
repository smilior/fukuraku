```markdown
## GET /users/:id

IDでユーザーを取得する。

**パラメータ:**
| 名前 | 型 | 必須 | 説明 |
|------|------|------|------|
| id | string | Yes | ユーザーID |

**レスポンス:**
- 200: ユーザーオブジェクト
- 404: ユーザーが見つからない
```

## コメントするべき場合

| ✅ コメントすべき | ❌ コメント不要 |
|----------------|--------------|
| なぜ（ビジネスロジック） | 何をしている（自明） |
| 複雑なアルゴリズム | 全ての行 |
| 非自明な挙動 | 自己説明的なコード |
| APIコントラクト | 実装の詳細 |

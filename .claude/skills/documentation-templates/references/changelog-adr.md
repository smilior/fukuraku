# Changelog・ADR テンプレート

## Changelog（Keep a Changelog 形式）

```markdown
# Changelog

## [Unreleased]
### Added
- 新機能

## [1.0.0] - 2025-01-01
### Added
- 初回リリース
### Changed
- 依存関係を更新
### Fixed
- バグ修正
```

## Architecture Decision Record（ADR）

```markdown
# ADR-001: [タイトル]

## Status
Accepted / Deprecated / Superseded

## Context
なぜこの意思決定を行うのか？

## Decision
何を決定したか？

## Consequences
トレードオフは何か？
```

## ADR番号付けルール

- `ADR-001` から連番
- ファイル名: `docs/adr/001-title.md`
- ステータス変更時はファイルを更新（削除しない）

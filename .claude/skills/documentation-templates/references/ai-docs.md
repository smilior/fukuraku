# AI対応・構造化ドキュメント テンプレート

## llms.txt テンプレート（AIクローラー向け）

```markdown
# プロジェクト名
> 一行の目的。

## Core Files
- [src/index.ts]: メインエントリー
- [src/api/]: APIルート
- [docs/]: ドキュメント

## Key Concepts
- コンセプト1: 簡潔な説明
- コンセプト2: 簡潔な説明
```

## RAGインデックス向けドキュメント設計

- H1〜H3の明確な階層構造
- データ構造はJSON/YAML例で示す
- フロー図はMermaidで記述
- 各セクションは独立して理解できるよう自己完結

## 構造化原則

| 原則 | 理由 |
|------|------|
| **スキャン可能** | ヘッダー・リスト・テーブル |
| **例を先に** | 見せて、語るだけでない |
| **段階的詳細** | シンプル → 複雑 |
| **最新維持** | 古い情報 = 誤解を招く |

> テンプレートは出発点。プロジェクトのニーズに合わせて適用すること。

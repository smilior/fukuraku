## 1. 「20万円の壁」の正体 - 収入ではなく「所得」

まず大前提として、「20万円の壁」は**収入**ではなく**所得**の話です。

**所得 = 収入（売上） - 経費**

多くの人が「副業で20万円稼いだら確定申告が必要」と思い込んでいますが、正確には「副業の**所得**が20万円を超えたら確定申告が必要」です。

### 具体例で理解する

**ケース1: 申告が必要**
- 副業の年間収入: 40万円（ライティング報酬）
- 経費: 10万円（パソコン周辺機器、書籍代）
- 所得: 40万円 - 10万円 = **30万円** → 確定申告が必要

**ケース2: 申告が不要（所得税のみ）**
- 副業の年間収入: 40万円（Web制作）
- 経費: 25万円（ソフトウェア代、サーバー代、外注費）
- 所得: 40万円 - 25万円 = **15万円** → 所得税の確定申告は不要

ケース2のように、収入が40万円でも経費を差し引けば所得は15万円。この場合、所得税の確定申告は不要です。**経費をしっかり記録・計上することがいかに重要かがわかります。**

### 法的根拠

この「20万円ルール」は、所得税法第121条第1項に定められています。給与所得者（年末調整を受けている会社員）で、給与所得以外の所得が20万円以下の場合、所得税の確定申告は不要とされています。

## 2. 所得の計算方法と経費の考え方

### 所得を正しく計算する手順

1. **年間の副業収入を合算する**: 複数の副業がある場合はすべて合算
2. **経費を集計する**: 副業に関連する支出をカテゴリ別に合計
3. **所得を算出する**: 収入 - 経費 = 所得

### 見落としがちな経費の例

副業サラリーマンが見落としがちな経費があります。きちんと計上すれば20万円以下に収まるケースも少なくありません。

| 経費カテゴリ | 具体例 | 年間の目安 |
|------------|--------|----------|
| 通信費（按分） | ネット回線、スマホ代の副業利用分 | 2〜5万円 |
| 地代家賃（按分） | 自宅の副業スペース分 | 6〜24万円 |
| 水道光熱費（按分） | 電気代の副業利用分 | 1〜3万円 |
| 消耗品費 | マウス、キーボード、文房具 | 1〜5万円 |
| 書籍費 | 技術書、参考書、オンライン講座 | 1〜5万円 |
| 交通費 | 打ち合わせ、コワーキングスペースへの交通費 | 1〜3万円 |

**按分の考え方**: 自宅兼事務所の場合、面積比や使用時間比で按分します。例えば、ワンルームで部屋の30%を作業スペースとして使っている場合、家賃の30%を経費にできます。

家賃8万円の場合、30%の按分で月2.4万円、年間28.8万円の経費に。これだけでも「20万円の壁」に対する経費としては大きな金額です。

## 3. 20万円以下でも申告が必要な3つのケース

「20万円以下だから何もしなくていい」は大きな間違いです。以下の3つのケースでは、所得が20万円以下でも申告が必要です。

### ケース1: 住民税の申告

**最も見落とされているのがこのケース**です。

「20万円ルール」は**所得税**にだけ適用されるルールです。**住民税**にはこのルールが適用されません。つまり、副業の所得が1円でもあれば、住民税の申告が必要です。

- 所得税 → 20万円以下なら確定申告不要
- 住民税 → 金額に関係なく申告が必要

住民税の申告は、お住まいの市区町村の役所で行います。確定申告書を提出した場合は住民税の申告は不要ですが、確定申告をしない場合は別途申告が必要です。

**申告しないとどうなる？** 自治体から問い合わせが来たり、後日まとめて請求される可能性があります。悪質と判断されれば加算税がかかることも。

### ケース2: 医療費控除やふるさと納税の還付申告

医療費控除やふるさと納税（ワンストップ特例を使わない場合）の還付申告を行う場合、副業の所得も合わせて確定申告する必要があります。

たとえ副業の所得が5万円だったとしても、医療費控除のために確定申告するなら、その5万円も申告書に記載しなければなりません。

### ケース3: 2箇所以上から給与を受けている場合

副業がアルバイトやパートで「給与所得」に該当する場合、年末調整を受けていない方の給与が20万円を超えると確定申告が必要です。

また、2箇所以上の給与の合計が一定額を超える場合など、条件によっては金額に関わらず確定申告が必要になるケースがあります。

## 4. 20万円を超えても確定申告で得するケース

確定申告は「面倒な義務」というイメージがありますが、実は**申告した方が得するケース**も多くあります。

### 源泉徴収されている場合の還付金

クラウドソーシングなどで報酬を受け取る際、10.21%が源泉徴収されていることがあります。

**例: 年間報酬50万円（源泉徴収後の手取り44.9万円）**
- 源泉徴収額: 51,050円
- 経費: 20万円
- 所得: 30万円
- 所得税額（概算）: 約15,000円
- 還付金: 51,050円 - 15,000円 = **約36,000円**

このケースでは、確定申告することで約36,000円が戻ってきます。**源泉徴収されている人は、確定申告した方がお得なケースが多いのです。**

### 赤字の場合の損益通算（事業所得の場合）

副業を「事業所得」として申告している場合、赤字が出たら給与所得と損益通算できます。つまり、副業の赤字分だけ給与にかかる税金が減ります。

ただし、「雑所得」の場合は損益通算できないため、事業所得として認められるための条件（継続性、反復性、事業としての実態）を満たしている必要があります。

## 5. 「20万円の壁」を活用した賢い副業戦略

「20万円の壁」を正しく理解した上で、賢い副業管理のポイントをまとめます。

### ポイント1: 経費を漏れなく記録する

経費を正しく計上すれば、所得を20万円以下に抑えられる可能性があります。「これは経費になるかな？」と迷ったら記録しておき、後で確認しましょう。

### ポイント2: レシートは撮影して即記録

レシートは放置すると紛失します。受け取ったらすぐにスマホで撮影し、アプリに記録する習慣をつけましょう。

### ポイント3: 月次で所得を把握する

年末に慌てないために、月ごとに収入と経費を集計し、所得の推移を把握しておきましょう。20万円に近づいているかどうかを早めに知ることが大切です。

### ポイント4: 20万円を超えそうなら開業届を検討する

所得が20万円を超えることが確実なら、開業届を出して青色申告にすることで最大65万円の控除を受けられます。経費と控除を合わせれば、税負担を大幅に減らせます。

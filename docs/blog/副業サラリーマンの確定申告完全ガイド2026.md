## 1. 副業サラリーマンに確定申告が必要な条件

### 基本ルール：副業の「所得」が年間20万円を超えたら確定申告が必要

会社員で副業をしている場合、副業の**所得**（収入から経費を引いた金額）が年間20万円を超えると、所得税の確定申告が必要です。これは所得税法第121条に基づくルールです。

ここでのポイントは「収入」ではなく「所得」であること。この違いを正しく理解することが、確定申告の第一歩です。

### 20万円以下でも申告が必要なケース

副業の所得が20万円以下でも、以下のケースでは申告が必要です。

- **住民税の申告**: 所得税の確定申告は不要でも、住民税の申告は必要。お住まいの市区町村に申告してください
- **医療費控除やふるさと納税の還付申告をする場合**: 他の理由で確定申告をする場合、副業の所得も合算して申告する必要があります
- **2箇所以上から給与を受けている場合**: 副業がアルバイトなどの「給与所得」に該当する場合は、金額に関わらず確定申告が必要なケースがあります

## 2. 「収入」と「所得」の違いを正しく理解する

確定申告で最も重要な概念が「収入」と「所得」の違いです。

**所得 = 収入 - 経費**

| 用語 | 意味 | 具体例 |
|------|------|--------|
| 収入 | 副業で得た売上の合計額 | クラウドソーシングの報酬50万円 |
| 経費 | 副業に必要な支出 | パソコン代、通信費、書籍代で合計35万円 |
| 所得 | 収入から経費を引いた金額 | 50万円 - 35万円 = 15万円 |

上記の例では、収入は50万円ですが、経費を差し引いた所得は15万円。所得が20万円以下のため、所得税の確定申告は不要です（住民税の申告は必要）。

**経費を正しく計上することで、確定申告が不要になるケースもあります。** 日頃から経費の記録をしっかり残しておくことが重要です。

## 3. 副業の所得区分：雑所得と事業所得

副業の所得は「雑所得」と「事業所得」のどちらかに分類されます。

| 項目 | 雑所得 | 事業所得 |
|------|--------|---------|
| 対象 | 一時的・小規模な副業 | 継続的・反復的な副業 |
| 青色申告 | 不可 | 可能（最大65万円控除） |
| 損益通算 | 不可 | 給与所得と通算可能 |
| 赤字の繰越 | 不可 | 3年間繰越可能 |
| 開業届 | 不要 | 必要 |

**判断基準:** 副業が継続的・反復的に行われており、事業としての実態がある場合は「事業所得」として申告できます。単発の収入や小規模な副業は「雑所得」として扱われることが一般的です。

副業の規模が大きくなってきたら、税務署に開業届を提出し、青色申告に切り替えることで最大65万円の控除を受けられます。

## 4. 確定申告に必要な書類一覧

確定申告の前に、以下の書類を準備しましょう。

### 必須書類

- **源泉徴収票**: 本業の会社から1月末頃に届きます
- **副業の収入がわかる書類**: 請求書、支払調書、振込記録など
- **経費の領収書・レシート**: 1年分をカテゴリ別に整理
- **マイナンバーカード**: e-Taxで電子申告する場合に必要
- **銀行口座情報**: 還付金の振込先

### あると便利な書類

- **各種控除の証明書**: 生命保険料控除証明書、ふるさと納税の受領書など
- **医療費の領収書**: 医療費控除を申請する場合

## 5. 確定申告の具体的なやり方（3ステップ）

### Step 1: 収入と経費を集計する

1年間の副業の収入と経費を集計します。

- 副業の売上を合算（請求書や振込記録から）
- 経費をカテゴリ別に整理・合算
- 所得（収入 - 経費）を計算

**ポイント:** この集計作業が確定申告で最も時間がかかる部分です。日頃からレシートを保存し、経費を記録しておくと大幅に時間を短縮できます。副楽のようなアプリを使えば、レシートを撮影するだけでAIが自動で分類・集計してくれます。

### Step 2: 確定申告書を作成する

以下のいずれかの方法で確定申告書を作成します。

- **国税庁の確定申告書等作成コーナー**: 無料で利用可能。画面の指示に従って入力するだけ
- **e-Tax（電子申告）**: マイナンバーカードがあればスマホからも申告可能
- **確定申告アプリ**: 副楽、freee、マネーフォワードなど

副業サラリーマンの場合、確定申告書は通常「確定申告書A」を使用します（事業所得の場合は確定申告書B）。

### Step 3: 申告書を提出する

作成した確定申告書を税務署に提出します。

| 提出方法 | メリット | デメリット |
|---------|---------|----------|
| e-Tax（電子申告）| 自宅から24時間提出可能 | マイナンバーカードが必要 |
| 郵送 | 税務署に行く必要なし | 届くまで数日かかる |
| 税務署に持参 | その場で確認してもらえる | 混雑する（特に3月） |

**提出期限: 毎年2月16日〜3月15日**

期限を過ぎると延滞税や無申告加算税が発生する可能性があるため、早めの準備をおすすめします。

## 6. 副業で経費にできるもの一覧

副業に関連する支出は経費として計上できます。主な経費は以下の通りです。

| カテゴリ | 経費の例 |
|---------|---------|
| 通信費 | インターネット回線（按分）、スマホ代（按分）、サーバー代、ドメイン代 |
| 消耗品費 | パソコン（10万円未満）、マウス、キーボード、文房具 |
| 書籍費 | 参考書、技術書、オンライン講座（Udemyなど） |
| 交通費 | 打ち合わせへの電車代、タクシー代 |
| 地代家賃 | 自宅の家賃（按分） |
| 水道光熱費 | 電気代（按分） |
| 外注費 | デザイン外注、翻訳外注など |
| 雑費 | コワーキングスペース代、名刺代 |

**「按分」とは**: 自宅で副業をしている場合、プライベートと仕事の使用割合に応じて経費を按分（あんぶん）します。例えば、家賃10万円で作業スペースが全体の20%なら、月2万円（年24万円）を経費として計上できます。

## 7. 会社に副業がバレない方法

副業が会社にバレる主な原因は**住民税**です。確定申告を行うと、副業の所得に応じた住民税が本業の会社に通知される場合があります。

**対策: 確定申告書の第二表で住民税を「自分で納付」を選択する**

確定申告書の第二表にある「住民税に関する事項」で、「自分で納付」にチェックを入れてください。これにより、副業分の住民税は会社を通さず自分で納付する形になります。

**注意点:**
- 自治体によっては「自分で納付」が適用されない場合があります
- 100%バレないとは言い切れません
- 不安な場合は税理士に相談することをおすすめします

## 8. 確定申告をしないとどうなる？ペナルティ解説

確定申告が必要なのに申告しなかった場合、以下のペナルティが課される可能性があります。

| ペナルティ | 内容 |
|-----------|------|
| 無申告加算税 | 納税額の15〜20%（50万円超の部分は20%） |
| 延滞税 | 納期限の翌日から年2.4%（2ヶ月超は年8.7%）※2026年の場合 |
| 重加算税 | 悪質な場合は35〜40% |

「知らなかった」は通用しません。副業の所得が20万円を超えている場合は、必ず確定申告を行いましょう。

なお、期限後でも自主的に申告すれば、無申告加算税が軽減される場合があります。申告を忘れていた場合は、できるだけ早く申告してください。

## まとめ

- 副業の**所得**（収入 - 経費）が20万円超で所得税の確定申告が必要
- 20万円以下でも**住民税の申告は必要**
- 経費を正しく計上すれば、申告不要になるケースもある
- 確定申告書は**e-Tax**で電子申告するのが便利
- 会社バレ防止は住民税を「**自分で納付**」に設定
- 申告しないと**無申告加算税・延滞税**のリスクあり

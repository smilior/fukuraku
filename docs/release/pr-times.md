## 開発背景

### 増加する副業サラリーマンと確定申告の課題

厚生労働省が副業・兼業の促進に関するガイドラインを策定して以降、副業を行うサラリーマンは年々増加しています。しかし、副業収入の確定申告に関しては以下の課題が存在しています。

- **経費管理の煩雑さ**: レシートの保管・手入力・分類に膨大な時間がかかる
- **知識の不足**: 「どの支出が経費になるか」「所得20万円の基準」など、税務知識が不足している方が多い
- **既存ツールのミスマッチ**: freeeやマネーフォワードなどの会計ソフトは、本格的なフリーランスや法人向けに設計されており、副業サラリーマンにはオーバースペック

副楽は、これらの課題を解決するために、副業サラリーマンに特化したシンプルな経費管理アプリとして開発されました。

## 主な特徴

### 1. AIによるレシート自動読取り・経費分類

スマートフォンでレシートを撮影するだけで、AIが日付・金額・店名・品目を読み取り、適切な経費カテゴリ（通信費、消耗品費、交通費など）に自動分類します。OpenAI GPT-4o Visionを活用し、読取精度94%を実現しています。

### 2. 副業所得20万円ライン自動監視

副業の所得（収入-経費）をリアルタイムで計算し、確定申告が必要となる年間20万円のラインに近づくとアラートで通知します。「確定申告が必要かどうか」を常に把握できるため、申告漏れを防止します。

### 3. 副業サラリーマン専用のシンプルUI

会計の専門知識がない方でも直感的に操作できるシンプルなインターフェースを提供します。既存の会計ソフトが複雑すぎると感じた方でも、迷うことなく経費管理を開始できます。

### 4. 確定申告に必要なデータをCSV出力

記録した経費データをCSV形式で出力でき、確定申告書への転記をスムーズに行えます。

## 技術情報

副楽は以下の技術スタックで構築されています。

- フロントエンド: Next.js 16（App Router）
- バックエンド・認証・データベース: Supabase（東京リージョン）
- AI: Vercel AI SDK + OpenAI GPT-4o Vision API
- 決済: Stripe
- ホスティング: Vercel
- PWA対応によるスマートフォンネイティブアプリに近い操作性

全データは東京リージョンのサーバーに保存され、Row Level Security（RLS）によるユーザーデータの完全分離を実現しています。

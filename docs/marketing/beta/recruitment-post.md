## note投稿文

### タイトル：「副業の確定申告に30時間かかった私が、3分で終わるアプリを作った話 - ベータテスター募集」

```markdown
# 副業の確定申告に30時間かかった私が、3分で終わるアプリを作った話

## はじめに

副業を始めて3年。
毎年、確定申告の季節になると憂鬱になっていました。

レシートの山をExcelに手打ちして、
「これは通信費？消耗品費？」と国税庁のサイトを何度も往復して、
気づけば30時間。

freeeやマネーフォワードも試しました。
でも、副業サラリーマンには高機能すぎて、
設定だけで挫折しました。

「副業の確定申告に特化した、シンプルなアプリがあればいいのに。」

誰も作らないなら、自分で作ろう。
そう思って開発を始めたのが「副楽（ふくらく）」です。

## 副楽でできること

副楽は、副業サラリーマン専用の確定申告支援アプリです。

### 1. レシートを撮るだけで経費入力

スマホでレシートを撮影するだけ。
AIが日付、金額、店名、品目を読み取り、
経費のカテゴリまで自動で分類してくれます。

### 2. 20万円ラインの自動監視

副業の所得が20万円を超えると確定申告が必要。
副楽はこのラインを常に監視して、
超えそうなタイミングで通知してくれます。

### 3. 確定申告に必要な書類を自動生成

経費の一覧表やCSVデータを自動で出力。
確定申告書への転記がスムーズになります。

## ベータテスター募集

現在、正式リリース前のベータテスト期間として、
テスターを募集しています。

**条件：**
- 副業で収入がある会社員の方
- 1〜2週間ほど使ってフィードバックをいただける方

**特典：**
- ベータ期間中は全機能を無料で利用可能
- 正式リリース後、有料プラン1ヶ月無料クーポン
- 開発者への直接フィードバック（機能リクエスト歓迎）

**応募方法：**
以下のフォームからご応募ください。
→ [Google Forms URL]

## おわりに

「副業の確定申告を、もう苦痛にしない。」

これが副楽のミッションです。

一人で開発しているので、
リアルなユーザーの声が何よりの力になります。

興味のある方、ぜひご参加ください。

---

副楽: https://fukuraku.smilior.com
お問い合わせ: fukuraku@smilior.com
```

## LP ベータ募集動線の改善提案

### 現状
LPのフッター付近にはベータ募集の導線がない。

### 改善案

#### 案1：フッターの上にCTAセクションを追加

```
--------------------------------------------
| 🎯 ベータテスター募集中                     |
|                                            |
| 正式リリース前に副楽を無料で体験できます。     |
| フィードバックをいただいた方には             |
| 有料プラン1ヶ月無料クーポンをプレゼント。     |
|                                            |
| [ベータテストに参加する]（CTA ボタン）       |
--------------------------------------------
```

#### 案2：ヒーローセクション直下にバッジ表示

LPのヒーローセクションの「無料で始める」ボタンの近くに
「現在ベータテスト中 - 全機能無料」のバッジを表示。
ベータ感を出すことで早期参加のインセンティブを高める。

#### 案3：フローティングバナー

ページスクロール時に画面下部に
「ベータテスター募集中 - 無料で全機能を体験」の
フローティングバナーを表示。クリックでGoogle Formsへ遷移。

### 推奨：案1 + 案2 の併用

理由：
- 案1でフッター到達者（＝興味を持って読んだ人）に訴求
- 案2でファーストビューでの認知を獲得
- フローティングバナーはUX低下のリスクがあるため初期は不採用

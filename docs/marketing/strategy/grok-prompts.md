# Grok 調査プロンプト集

フォロー対象リスト拡充のためのGrok質問テンプレート。
調査結果は `follow-list.md` に追記する。

---

## 未調査カテゴリ

### YouTube副業系（X連動）

```
副業・確定申告・節税をテーマにしたYouTubeチャンネルで、
Xアカウントも運営しているものをフォロワー数が多い順に10個教えてください。

条件：
- サラリーマンの副業・確定申告・節税を扱っている
- YouTubeとXを両方運営している
- Xのフォロワー数も可能な範囲で教えてください
```

---

### Podcast・Voicy系（X連動）

```
副業・お金・確定申告をテーマにしたPodcastまたはVoicyチャンネルで、
Xアカウントも運営しているものをフォロワー数が多い順に10個教えてください。

条件：
- サラリーマンの副業・節税・資産形成を扱っている
- XのアカウントIDも教えてください
- フォロワー数も可能な範囲で
```

---

### メディア・ニュース系（取材・掲載狙い）

```
副業・フリーランス・確定申告を特集することが多いWebメディアや
ニュースアカウントをXフォロワー数が多い順に10個教えてください。

例：東洋経済オンライン、プレジデントオンライン、ダイヤモンド・オンライン等
副業・個人開発・SaaS系メディアも含めてください。
XのアカウントIDと媒体の特徴も教えてください。
```

---

### 副業×女性インフルエンサー

```
副業・確定申告・節税をテーマに発信している女性インフルエンサーのXアカウントを
フォロワー数が多い順に10個教えてください。

条件：
- サラリーマン・OL向けの副業・節税コンテンツ
- XのアカウントIDとフォロワー数
- Note・ブログ・YouTubeとの連動有無も教えてください
```

---

### 個人開発×副業（競合・連携候補）

```
個人開発でSaaSや副業支援ツールを作っているエンジニアのXアカウントを
フォロワー数が多い順に10個教えてください。

条件：
- 確定申告・税務・マネー系のプロダクトを作っている、または関心が高い
- Next.js / Supabase / Stripe などのスタックを使っている
- XのアカウントIDとフォロワー数
```

---

## 使い方

1. 上のプロンプトをそのままGrokにコピペ
2. 調査結果をこのチャットに貼る
3. `follow-list.md` に自動追記

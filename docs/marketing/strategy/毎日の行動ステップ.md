# 副楽 毎日やるべき行動ステップ

> 開発とマーケティングを同時並行で進めるための、具体的な日次・週次ルーティン

---

## 大原則：1日2時間で回す

```
朝30分  → マーケティング（X投稿・ブログ・コミュニティ）
夜90分  → 開発（コーディング）
```

副業として続けるため、**毎日完璧にやらない**前提で設計する。
「今日できたか」ではなく「今週できたか」で判断する。

---

## 毎日のデイリールーティン

### 🌅 朝（30分）※通勤中でも可

| 時間 | 行動 | 具体的な内容 |
|------|------|------------|
| 5分 | Xの確認・リプライ | 前日の投稿へのコメントに返信。副業・税金系のツイートに共感リプ |
| 10分 | X投稿を作る | 曜日別テーマ（下記参照）に沿って1投稿を作り、投稿or予約 |
| 15分 | インプット | 副業・税金・個人開発系のXをROM。ネタを3つメモする |

### 🌙 夜（90分）※帰宅後

| 時間 | 行動 | 具体的な内容 |
|------|------|------------|
| 5分 | 今日のタスクを確認 | Week別の開発タスク表を見て、「今夜やること」を1つ決める |
| 75分 | **開発（コーディング）** | 決めた1タスクだけに集中。Cursor + Claude Codeで進める |
| 10分 | 進捗メモ | 今日やったこと・詰まった点・明日やることを3行でメモ |

## X（Twitter）曜日別投稿テーマ

> コンセプト：「副業サラリーマンの税金の悩みを解決する個人開発者」

| 曜日 | テーマ | 投稿例 |
|------|--------|--------|
| **月曜** | 副業×税金Tips | 「副業で月5万稼いだ時の手取りは？計算してみた」 |
| **火曜** | 開発の進捗報告 | 「AI仕訳の精度が87%→94%に上がった。改善したこと↓」 |
| **水曜** | 副業あるある・共感系 | 「確定申告の季節、freeeを開いて閉じた人RT🙋」 |
| **木曜** | Q&A・ユーザーの声 | 「よくある質問：副業の交通費は経費にできる？答え↓」 |
| **金曜** | 個人開発の裏話 | 「個人開発でMRR10万円を目指す記録 Day○ 今週やったこと」 |
| **土曜** | まとめ・役立ち情報 | 今週の副業税金Tipsをまとめ投稿。バズ狙いより保存狙い |
| **日曜** | お休み or ゆるい投稿 | 進捗グラフや作業風景など。無理しない |

**投稿のコツ：**
- 長文よりも「数字・リスト・Before/After」が伸びる
- 朝7:30 または夜20:00 に投稿（通勤・帰宅タイムに合わせる）
- 毎日投稿できない日は「今日はお休みです」の一言でOK

## 最初の1週間だけやること（Day 1〜7）

明日から始めるための、最小限のアクション：

1. **今夜：** Vercel / Supabase / Stripe / GitHub アカウントを作る（30分）
2. **今夜：** Xアカウントを開設して、1投目を投稿する
   > 「副業の確定申告が面倒すぎるので、アプリを作ります。進捗を毎日ツイートします」
3. **明日朝：** ドメインを取得する（fukuraku.app など、約1,500円）
4. **明日夜：** Next.js + Supabase テンプレートをクローンして、ローカルで動かす

**これだけで「始まった」状態になる。あとは毎日少しずつ進めるだけ。**

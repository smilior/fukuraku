# 副楽 マーケティング

## フォルダ構成

```
docs/marketing/
├── README.md            # このファイル
├── strategy/            # 戦略・準備ドキュメント
├── x/
│   ├── profile.md       # プロフィール設定・ピン留め文案
│   ├── drafts/          # 下書き
│   └── published/       # 投稿済みアーカイブ
└── note/
    ├── drafts/          # 下書き
    └── published/       # 公開済みアーカイブ
```

## 運用ルール

- ファイル名: `YYYY-MM-DD_内容メモ.md`
- 投稿したら `drafts/` → `published/` に移動
- フォロー対象の追加・更新は `strategy/follow-list.md`

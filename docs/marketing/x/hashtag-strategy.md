# ハッシュタグ戦略

## 基本ルール

- 1投稿あたり1〜2個まで（多すぎるとスパム扱い・リーチ低下）
- タグは投稿本文の最後に配置
- 投稿内容に直接関連するタグのみ使用

---

## メインハッシュタグ（常用）

| ハッシュタグ | 用途 | 使用頻度 |
|------------|------|---------|
| #副業 | 副業全般の投稿 | 高 |
| #確定申告 | 確定申告Tips・Q&A | 高 |
| #個人開発 | 開発進捗・技術の話 | 中 |

## 避けるべきタグ・注意事項

### 使わないタグ
- `#フォロバ100` 等の相互フォロー系 → スパムアカウントが集まるだけ
- `#拡散希望` → 効果が薄く、ブランドイメージを損なう
- `#PR` `#広告` → ステマ規制対応が必要な場合のみ使用

### 注意事項
- トレンドタグに便乗する場合は、内容が関連している場合のみ
- 政治・宗教・炎上中のタグには絶対に便乗しない
- 同じタグの連投は避ける（1日同じタグは2回まで）

---

## タグの効果測定

月1回、以下の指標でタグの効果を確認する。

| 指標 | 確認方法 |
|------|---------|
| インプレッション | X Analytics でタグ別に比較 |
| エンゲージメント率 | いいね+RT+リプライ / インプレッション |
| フォロワー増減 | タグ使用時の新規フォロワー数 |
| 流入 | Google Analytics で X からの流入を確認 |

効果の高いタグは頻度を上げ、効果の低いタグは入れ替える。

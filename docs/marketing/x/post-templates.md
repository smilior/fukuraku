## カテゴリ2：Q&A形式

### テンプレート7
```
Q. 副業の収入が20万円以下なら確定申告は不要？

A. 所得税の確定申告は不要です。
ただし、住民税の申告は必要です。

これ、意外と知らない人多い。
住民税の申告をしないと、
自治体から連絡が来ることも。

市区町村の窓口で簡単に申告できます。

#副業確定申告
```

### テンプレート8
```
Q. 家賃って経費にできる？

A. 自宅で副業してるなら「按分」で一部経費にできます。

例：
家賃10万円
作業スペース：部屋全体の20%
→ 月2万円が経費

按分の根拠は
- 面積比
- 使用時間比
のどちらかで計算。

#副業 #経費管理
```

### テンプレート9
```
Q. クラウドソーシングの源泉徴収って何？

A. 報酬から事前に税金が引かれる仕組み。

例：報酬10万円の場合
源泉徴収額：10,210円
手取り：89,790円

確定申告すると、
この源泉徴収分が還付される可能性あり。

特に経費が多い人は確定申告した方がお得。

#副業確定申告
```

### テンプレート10
```
Q. 副業で赤字が出た場合はどうなる？

A. 所得の種類によります。

- 雑所得の赤字 → 他の所得と損益通算できない
- 事業所得の赤字 → 給与所得と損益通算可能

つまり、事業所得にできれば
副業の赤字で給与の税金が減る可能性あり。

ただし「事業所得」と認められるには
継続性・反復性が必要。

#副業
```

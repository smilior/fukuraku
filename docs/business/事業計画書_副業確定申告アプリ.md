## エグゼクティブサマリー

### ビジネスの概要

「副楽」は、副業収入がある会社員のために作られた、国内唯一の確定申告専用アプリです。

- **ターゲット:** 副業サラリーマン（推計180〜250万人）
- **課題:** freee・マネーフォワード等の会計ソフトは機能が多すぎて使えない
- **解決策:** 副業の雑所得だけに特化した、3ステップで完了するシンプルなアプリ
- **価格:** 月額980円（競合の半額以下）

### 数字で見るビジネスチャンス

| 指標 | 数値 |
|---|---|
| 副業実施中の会社員 | 180〜250万人 |
| うち確定申告が必要な層 | 100〜150万人 |
| 市場規模（SAM） | 60〜80億円 |
| 競合の価格 | 月額900〜980円（freee・マネーフォワード） |
| 本アプリの目標価格 | 月額980円（年払い817円/月相当） |
| 月10万円達成に必要なユーザー数 | 有料91人 |

### 12ヶ月の目標

- **初期費用:** 約1,500円（ドメイン代のみ）

**CalQ Shareとの決定的な差**
CalQ Share（最も近い競合）は無料だが、**AI仕訳なし・e-Tax非対応・更新頻度不明**。「使えるが不安」な選択肢であり、月980円でも副楽が選ばれる余地は十分。

### 2-2. ポジショニングマップ

```
                    機能が豊富・汎用的
                         ↑
           マネーフォワード | freee
                    ★     |   ★
         やよい（青色）     |
               ★          |
 副業に非特化 ─────────────+──────────── 副業に特化
                          |    タックスナップ
               やよい（白色）       ★
                 ★        |
                          |   CalQ Share
                          |      ★
                          |
                          |  ★★★ 副楽 ★★★
                          | （最もシンプル×最も副業特化）
                         ↓
                    シンプル・特化型
```

### 2-3. 選ばれる理由3つ

1. **圧倒的シンプルさ** — 会計用語ゼロ。「収入」「経費」だけで完結。15分で申告書完成
2. **業界最安水準** — 年払い9,800円（月817円相当）。freeeの年額の約83%
3. **副業サラリーマンだけの機能**
   - **20万円超えアラート**（他社にない独自機能）
   - **会社バレ防止チェック**（住民税の普通徴収切替えをガイド）
   - **年末調整との整合チェック**（源泉徴収票と副業所得の合算確認）

## 第4章：プロダクト計画

### 4-1. MVPの機能スコープ

**判断基準:** 以下3つの問いをすべてクリアした機能のみ実装
1. 副業サラリーマンが20万円超の雑所得を申告するために必須か？
2. 1人開発で3〜4週間以内に実装可能か？
3. 競合との差別化に直結するか？

**MVP必須機能（Must Have）**

| 機能 | 説明 |
|---|---|
| メール認証 + Googleログイン | ユーザー管理の基盤 |
| 副業収入の手動記録 | 日付・金額・取引先・種別（源泉徴収対応） |
| 経費の手動記録 | 日付・金額・カテゴリ・メモ |
| **レシート撮影 → AI仕訳** | 最大の差別化。GPT-4o Visionで1枚約0.2円 |
| **20万円超アラート** | リアルタイム所得計算。15万円で黄色、20万円で赤色 |
| ダッシュボード | 収入・経費・所得の推移グラフ。20万円バー表示 |
| 確定申告サマリー + CSVエクスポート | 年間収支の一覧。国税庁サイトへの連携用 |
| オンボーディング | 副業タイプ選択（ライティング・物販・シェアリング等） |
| 設定画面 | プロフィール・通知・データ管理 |

**後回し機能（v1.1以降）**
- 銀行口座・クレカ自動連携
- 確定申告書PDF自動生成
- e-Tax連携

**やらない機能（Out of Scope）**
- 青色申告（複式簿記）
- 消費税申告
- 仮想通貨・FXの損益計算
- 法人向け機能

### 4-2. 主要画面の構成

```
[オンボーディング] → [ダッシュボード（ホーム）]
                           ↙         ↘
              [収入記録]          [経費記録]
                                 ↙
                    [レシート撮影 → AI仕訳]
                           ↓
                  [20万円アラート画面]
                           ↓
               [確定申告サマリー + CSVエクスポート]
```

**ダッシュボードの核心 — 20万円バー:**
```
今年の副業所得: ████████░░  16.5万円 / 20万円
                            あと3.5万円で確定申告が必要です
```

### 4-3. 技術スタック（バイブコーディング最適化）

| カテゴリ | 選定技術 | 理由 |
|---|---|---|
| フロントエンド | **Next.js 15 + shadcn/ui + Tailwind** | Cursorとの相性◎。PWAでiOS/Android両対応 |
| バックエンド・DB | **Supabase** | Auth + DB + Storage + Functions が一体。無料枠充実 |
| AI（レシートOCR） | **OpenAI GPT-4o Vision API** | 1枚約0.2円。Structured Outputで精度向上 |
| 決済 | **Stripe Checkout + Customer Portal** | コード最小でサブスク管理 |
| ホスティング | **Vercel** | Next.jsの自動デプロイ。無料枠で開始 |
| 開発ツール | **Cursor Pro + Claude Code** | バイブコーディングの中核 |

**スタート時のコスト:**
```
開発期間（ユーザー0人）: 月約2,000円（ドメイン代のみ）
運用開始後（〜100人）: 月約4,000〜7,000円
```

**出発点:** Vercel公式の Next.js Subscription Payments テンプレートを使えば、認証・決済・DBの設定を大幅省略できる

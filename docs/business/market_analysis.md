## 1. 市場規模分析

|------|------|------|
| 副業がある就業者数（実数） | 約305万人 | 総務省「令和4年就業構造基本調査」(2022) |

**推計：2025〜2026年の副業実施人口**

- 正社員数は約3,600万人（総務省統計ベース）
- 副業実施率が2022年の2.5%から2025年は約5〜7%に上昇と推計（Job総研の「4割が副業経験あり」を踏まえ、現在進行形で実施中の割合を推計）
- **副業を現在実施中の会社員：推計180万〜250万人（2025年時点）**
- このうち副業所得が年20万円を超え確定申告が必要な層：推計100万〜150万人

- 市場規模：**約500〜600億円**（ソフト利用料＋税理士報酬のデジタル代替分含む）

### 1-5. 副業市場の成長率・トレンド（2025〜2026年）

**成長ドライバー:**

1. **政府の副業推進政策**: 厚生労働省「副業・兼業の促進に関するガイドライン」改定。2027年度以降「希望者全員が副業可能な社会」を目標に設定。
2. **企業の副業解禁加速**: 大企業の82.4%が副業に関する社内規程を整備済み（2025年調査）。
3. **副業実施率の急増**: 2022年→2025年で副業経験者が約2割増加（Job総研調査）。
4. **税制改正の追い風**: 2026年の基礎控除引き上げ等で「確定申告」への関心が高まっている。
5. **リモートワークの定着**: コロナ後もリモート勤務が定着し、副業しやすい環境が維持。

**市場成長率の推計:**
- 副業人口の年間成長率：**15〜20%**（2022年305万人 → 2025年推計180〜250万人の実施中人口）
- クラウド会計ソフト市場のCAGR：**9〜10%**（グローバル）
- 副業×確定申告アプリ市場の成長率：**年20〜25%**（副業人口増 × デジタル化率上昇の複合効果）

### 2-2. 各競合の強み・弱み分析

#### freee確定申告
- **強み**: 簿記知識不要のUI設計、豊富な外部連携（POSレジ・ECサイト等）、知名度・ブランド力
- **弱み**: 個人向けは月額980円〜と高め。機能が豊富すぎて副業だけの人にはオーバースペック。スマホだけでは完結しにくい。

#### マネーフォワード クラウド確定申告
- **強み**: 銀行・カード連携数が業界最多。複式簿記ベースで税理士との連携がスムーズ。経理の自動化に強い。
- **弱み**: 会計知識がある前提のUI。副業初心者には敷居が高い。レシート撮影に月間件数制限あり。

#### やよいの青色申告オンライン / 白色申告オンライン
- **強み**: 個人向けシェアNo.1（55.4%）。白色申告は無料。初年度無料キャンペーンで導入しやすい。長年の実績と信頼性。
- **弱み**: UIが古い印象。スマホ対応が遅れている。副業特化の機能はない。PC操作前提。

#### タックスナップ (Taxnap)
- **強み**: スマホ完結、スワイプで簡単仕訳、AI「丸投げ仕分け」が革新的。副業の雑所得の確定申告は無料対応。
- **弱み**: 新興サービスで知名度が低い。主にフリーランス・個人事業主向けで副業サラリーマンに完全特化ではない。

#### CalQ Share（カルクシェア）
- **強み**: 会社員の副業に特化した唯一の無料アプリ。漫画での確定申告解説。完全無料。
- **弱み**: 機能が限定的（AI仕訳なし、e-Tax非対応）。UIの洗練度が低い。アップデート頻度が不明。マネタイズ不透明で継続性にリスク。

### 2-3. 競合のポジショニング上の空白

**既存競合が埋めていないニーズ:**
1. 「副業の雑所得だけ」に完全特化したシンプルなUX → freee/MF/弥生はオーバースペック
2. 月額300〜500円の低価格帯 → freee 980円、MF 900円に比べ半額以下
3. 「20万円超えアラート」のような副業特有の機能 → 既存ツールにはない
4. 会社の年末調整との連携を意識した設計 → 既存ツールは個人事業主前提

## 3. ポジショニング

### 3-1. 競合との差別化マップ

```
                    機能が豊富・汎用的
                         ↑
                         |
           マネーフォワード  |  freee
              ★           |    ★
                         |
                  やよい（青色）
                    ★     |
    副業に非特化 ←--------+--------→ 副業に特化
                         |
              やよい（白色）|     タックスナップ
                ★        |        ★
                         |    CalQ Share
                         |      ★
                         |
                         |   ★★★ 当アプリ ★★★
                         |  （最もシンプル × 最も副業特化）
                         ↓
                   シンプル・特化型
```

**当アプリのポジション**: 右下の「シンプル × 副業特化」の象限に位置。
既存の大手3社（freee、マネーフォワード、弥生）はいずれも「汎用的」で、副業サラリーマンにとっては機能過多。CalQ Shareは副業特化だが機能不足。当アプリはこの間を埋める最適なポジション。

### 3-2. 「選ばれる理由」3つ

#### 理由1: 圧倒的なシンプルさ ── 「3ステップで確定申告完了」
- 副業の雑所得に完全特化することで、不要な機能（複式簿記、消費税申告、棚卸し等）を全て排除
- レシート撮影 → AI自動仕訳 → 確定申告書PDF出力の3ステップ
- 初めての確定申告でも、15分で完了できるUXを目指す

#### 理由2: 業界最安水準の月額300〜500円
- freee（980円）やマネーフォワード（900円）の半額以下
- 副業の年間所得が数十万円の層にとって、年間3,600〜6,000円の利用料は十分にペイする
- 「確定申告の不安を解消する保険」として、月ワンコインの心理的ハードルの低さ

#### 理由3: 副業サラリーマンだけの「あったらいいな」機能
- **20万円超えアラート**: 副業所得が20万円に近づくとプッシュ通知。確定申告の要否を自動判定
- **会社バレ防止チェック**: 住民税の普通徴収切替えなど、会社に副業がバレないための設定をガイド
- **年末調整との整合チェック**: 会社の源泉徴収票と副業所得を合算し、正確な申告をサポート

### 3-3. 参入タイミングの優位性

1. **副業人口の急拡大期（2025〜2027年）**: 政府目標「2027年度に希望者全員が副業可能」に向け、副業人口は今後2〜3年で急拡大が見込まれる。この「第一波」のユーザーを獲得できるのは今しかない。

2. **確定申告のデジタル化加速**: e-Taxの普及率向上、マイナンバーカードの普及（2025年時点で保有率約80%）、スマホ申告の整備が進み、アプリでの確定申告が「当たり前」になりつつある。

3. **大手が副業特化に本腰を入れる前の参入**: freee・マネーフォワードは法人向けに注力しており、「副業サラリーマン専用」のような小さなセグメントに経営リソースを割く優先度は低い。大手が本格参入する前にブランドを確立できる。

4. **AI技術の成熟**: レシートOCR・自動仕訳のAI精度が実用レベルに到達。個人開発・小規模チームでも高品質なAI機能を実装可能な環境が整った。

## 4. ターゲットユーザー詳細（ペルソナ）

### ペルソナA: 副業初心者サラリーマン「田中 健太（32歳）」

| 項目 | 詳細 |
|------|------|
| **年齢・性別** | 32歳・男性 |
| **職業** | IT企業の営業職（正社員・年収500万円） |
| **副業** | Webライティング（クラウドソーシング経由）、始めて1年目 |
| **副業収入** | 月3〜5万円（年間36〜60万円） |
| **ITリテラシー** | スマホは使いこなすが、会計知識はゼロ |
| **家族構成** | 妻と2歳の子ども |

**課題（ペイン）:**
- 「確定申告」という言葉だけで不安になる。何をすれば良いのか全く分からない
- 副業所得が20万円を超えたのかどうか、そもそも正確に把握できていない
- freeeを試したが、「勘定科目」「借方/貸方」の意味が分からず挫折
- 会社に副業がバレるのが怖い（住民税の仕組みを理解していない）
- 平日は忙しく、確定申告のために時間を取りたくない

**ジョブ To Be Done:**
- 「副業の確定申告を、最小限の手間と知識で、間違いなく終わらせたい」
- 「会社にバレずに、合法的に副業の税金を処理したい」
- 「領収書やレシートの管理をラクにしたい（溜め込んでしまう）」

**当アプリが解決する方法:**
- 会計用語を一切使わないUI（「売上」「経費」だけで完結）
- レシートを撮影するだけでAIが自動仕訳
- 20万円超えアラートで確定申告の要否を自動通知
- 「会社バレ防止チェック」で住民税の処理をガイド
- 確定申告シーズンに15分で申告書PDF完成

### ペルソナB: 複数副業を持つ「鈴木 美咲（38歳）」

| 項目 | 詳細 |
|------|------|
| **年齢・性別** | 38歳・女性 |
| **職業** | 大手メーカーの経理部（正社員・年収600万円） |
| **副業** | (1) メルカリでのハンドメイド販売 (2) 週末のヨガインストラクター (3) ブログのアフィリエイト収入 |
| **副業収入** | 月8〜15万円（年間96〜180万円） |
| **ITリテラシー** | 高い（仕事で会計ソフトも使用） |
| **家族構成** | 独身 |

**課題（ペイン）:**
- 3つの副業の収支管理が煩雑。メルカリ・銀行振込・現金払いが混在
- 経理の知識はあるが、本業の会計ソフト（弥生）で副業まで管理したくない
- 副業の種類ごとに「事業所得」か「雑所得」かの判断に迷う
- 年間100万円を超える副業収入があるため、経費をしっかり計上したい
- 複数の副業の確定申告を一つのアプリで完結させたい

**ジョブ To Be Done:**
- 「複数の副業の収支を一元管理し、確定申告をスムーズに完了したい」
- 「経費をきちんと計上して、税金を最適化したい」
- 「事業所得か雑所得かの判定を、アプリがガイドしてほしい」

**当アプリが解決する方法:**
- 副業ごとにカテゴリ分けして収支を管理できるUI
- メルカリ・PayPayなど複数の入金元を一括連携
- レシート撮影AIで経費の仕訳を自動化（複数副業を横断）
- 「事業所得 or 雑所得」の簡易判定フロー
- 収入が大きい場合は「青色申告への切替えを検討」と提案（上位プランまたは他サービスへの誘導）
